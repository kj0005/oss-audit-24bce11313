## Project Description
This project presents an Open Source Audit of Git, a widely used distributed version control system. It analyzes Git’s origin, development, GPL license, open-source philosophy, and ethical aspects. The project also explores Git’s Linux footprint, ecosystem, and comparison with proprietary alternatives. Additionally, it includes five shell scripts demonstrating basic Linux commands, system analysis, and automation techniques.

## Scripts Included

### Script 1: System Identity Report
Displays system information such as Linux distribution, kernel version, logged-in user, uptime, and date.

### Script 2: FOSS Package Inspector
Checks whether Git is installed on the system and displays its version and related details.

### Script 3: Disk and Permission Auditor
Analyzes important system directories and displays their permissions and disk usage.

### Script 4: Log File Analyzer
Reads a log file and counts occurrences of a keyword such as "error" and displays summary results.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source philosophy statement based on user input and saves it to a file.
